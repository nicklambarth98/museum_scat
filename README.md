# museum_scat
Contains files for the Kaggle Competition MuseumSCAT Specimen Collection Annotation Task

In this project I aimed to develop a greater understanding of the following: 
- Image preprocessing
- Optical Character Recognition (OCR)
- PyTesseract
- OpenCV
- Calling VLMs using Ollama
- Heuristic OCR

The work is split into two notebooks: 
1. **EDA and VLM** - Containing EDA for the project across some key information, preprocessing, and classification using a pre-trained VLM.
2. **Heuristic OCR** - Containing pre-processing and an Heuristic method of OCR and classification for this task.

- CNN ensemble methods for classification with the goal of using boosting (data limitation).
